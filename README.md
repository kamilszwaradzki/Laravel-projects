## petstore-api

### Przeznaczenie \ Purpose
**EN:** An application in Laravel 12 that acts as a frontend for the PetStore public API. It allows you to perform all CRUD operations (create, read, update, delete) on the `/pet` resource through a user-friendly, form-based GUI. Robust error handling and presentation for the end user is provided.\
**PL:** Aplikacja w Laravel 12, która działa jako frontend do publicznego API PetStore. Umożliwia wykonanie wszystkich operacji CRUD (dodaj, przeglądaj, edytuj, usuń) na zasobie `/pet` poprzez przyjazne, formularzowe GUI. Zapewniona solidna obsługa i prezentacja błędów dla użytkownika końcowego. \
[link](https://github.com/kamilszwaradzki/petstore-api)
## smart-brief-ai
![smart_brief_generator_png](https://github.com/kamilszwaradzki/smart-brief-ai/raw/main/smart_brief_generator.png)
### Przeznaczenie \ Purpose
**EN:** An application written in Laravel 12 that automates the initial stage of project planning. The system analyzes unstructured text provided by the user (e.g., an email from a customer, notes from a conversation, a description of requirements) and uses a language model (LLM) to generate a structured working document containing: 1) a summary, 2) tasks, 3) questions, 4) risks. \
**PL:** Aplikacji napisana w Laravel 12, automatyzująca początkowy etap planowania projektu. System analizuje dostarczony przez użytkownika nieuporządkowany tekst (np. email od klienta, notatki z rozmowy, opis wymagań) i wykorzystuje model językowy (LLM) do wygenerowania ustrukturyzowanego dokumentu roboczego zawierającego: 1) podsumowanie, 2) zadania, 3) pytania, 4) ryzyka. \
[link](https://github.com/kamilszwaradzki/smart-brief-ai)
